# Решение квадратных уравнений на C++

## Описание

Программа для решения квадратных уравнений с поддержкой **длинных чисел** (до 100000 знаков) и валидацией ввода через паттерн **«Интерпретатор»**.

## Возможности

- Решение полных и неполных квадратных уравнений
- Обработка вырожденных случаев (INF, NO SOLUTION, WRONG)
- Поддержка чисел с точностью до 10^5 десятичных знаков
- Валидация входных данных (паттерн Интерпретатор)
- Модульные тесты (Google Test)
- Полностью на русском языке (вывод и документация)

## Формат вывода (по спецификации)

- `OK` — есть решение(я)
  - следующая строка: количество корней
  - затем корни (каждый на новой строке)
- `INF` — бесконечно много решений
- `WRONG` — неверный ввод коэффициентов
- `NO SOLUTION` — нет решений

## Требования

- Компилятор с поддержкой C++17
- CMake 3.10+
- Google Test (для запуска тестов)
